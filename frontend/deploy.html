6 Setup MetaMask

  1. Install MetaMask browser extension
  2. Click MetaMask icon ‚Üí Settings ‚Üí Networks ‚Üí Add Network

  4. Click "Save"

7 Import Ganache Accounts

  3. Repeat for multiple accounts (to test different roles)
‚ÑπÔ∏è The first account (0) will be the default deployer with Admin role

Troubleshooting

Problem: "Could not connect to node at http://127.0.0.1:8585"
Solution: Make sure Ganache is running in a separate terminal (Step 1)
Problem: MetaMask shows "Unrecognized Chain"
Solution: Make sure you added the network with Chain ID 1337 (Step 6)
Problem: "Please connect MetaMask first" message
Solution: Click the "Connect MetaMask" button and approve the connection
Problem: Functions not working after contract update
Solution: Update CONTRACT_ADDRESS in config.js with the new deployed address
